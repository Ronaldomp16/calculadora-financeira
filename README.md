# 📊 Calculadora Financeira

Projeto em Python com Streamlit para simulação de investimentos.

## Funcionalidades

- Juros simples
- Juros compostos
- Juros com aporte mensal
- Comparação com inflação

## Tecnologias

- Python
- Streamlit
- Matplotlib
- Requests

## Como rodar

```bash
pip install -r requirements.txt
streamlit run app.py
